1. Overview

SmartPM is a full-featured ELV (Extra-Low Voltage) construction project management application. It handles everything from importing AI-generated estimates from SmartPlans to day-to-day tracking of materials installed, labor hours spent, billing, change orders, RFIs, and infrastructure.

Key Features

FeatureDescription
📦 SmartPlans ImportOne-click import of AI-generated estimates with locked budgets
📐 Work Breakdown (WBS)Hierarchical phase → location → task tracking from the bid
🏢 InfrastructureMDF/IDF/TR room management with equipment, cable runs, and labor
🚦 Budget HealthTraffic light system (🟢🟡🔴) for instant budget visibility
📋 Schedule of ValuesAIA G703 line items for progress billing
💰 Progress BillingTrack payment applications per billing period
📝 Change OrdersDocument scope changes with cost and schedule impact
❓ RFIsTrack Requests for Information through resolution

📧 Daily ReportGenerate emailable daily status reports for stakeholders
✅ Punch ListDeficiency tracking through closeout

ℹ️ All data is stored securely in the cloud (Cloudflare D1 database). Projects persist across sessions and can be accessed from any device.

4. Dashboard

The Dashboard is your home screen. It shows all projects with summary metrics.

Dashboard Metrics

  • Active Projects — Currently in progress
  • Total Contract Value — Sum of all project contracts
  • Total Billed — Sum of all billing to date
  • Open RFIs — Unresolved questions
  • Pending COs — Change orders awaiting approval
  • Outstanding — Amount billed but not yet paid

Project Cards

Each project appears as a card showing name, status, contract value, billed amount, and disciplines. Click a card to open the project.

5. Importing from SmartPlans

This is the primary workflow for creating projects.

  1. On the Dashboard, click "📦 Import from SmartPlans"
  2. Click "📁 Choose JSON File" and select the SmartPlans export
  3. Review the import preview (project name, disciplines, RFIs)
  4. Click "✓ Import Project"
  5. SmartPM creates the project with all data pre-populated

What Gets Imported

SmartPlans DataSmartPM Destination
Project name, type, locationProject metadata
Pricing configurationProject settings
Selected RFIsRFIs module
SOV line itemsSchedule of Values
MDF/IDF roomsInfrastructure locations
Equipment per roomLocation items (🔒 locked budgets)
Cable runsCable run records
WBS phases & tasksWork Breakdown Structure
💡 Budgeted amounts imported from SmartPlans are locked (🔒). Only Admins/Ops Mgrs can modify budgets. PMs can only update actual quantities and hours.

6. Work Breakdown Structure (WBS)

The WBS shows your entire project broken down into Phases → Locations → Tasks.

WBS Dashboard

  • Project Health — Overall traffic light (🟢🟡🔴)
  • Total Tasks — Number of WBS items
  • Overall Progress — Weighted % complete
  • Material Budget — Actual vs. budgeted material costs
  • Labor Budget — Actual vs. budgeted labor hours
  • Tasks Complete — Completed task count

Editing a WBS Task

Click the ✏️ button on any task to open the update modal:

SectionWhat You See
📊 Estimated from Bid (Read-Only)Material cost, labor hours, and total budget from the original SmartPlans estimate
✏️ Actual ProgressEnter quantity installed (ft or ea), actual labor hours, progress %, and status
💡 PMs only enter what they know: how many feet or pieces were installed and how many hours it took. You do NOT need to enter dollar amounts — the system calculates costs automatically.

9. Schedule of Values & Billing

SOV

AIA G703-format line items. Each line item has a scheduled value, material/labor breakdown, and completion percentage. Items are auto-created from SmartPlans import.

Progress Billing

Create billing periods (monthly) and record work completed and materials stored for each SOV line item. The system calculates retainage, previous payments, and current payment due.

10. Change Orders

Track scope changes with financial and schedule impact.

  • Types: Addition, Deduction, No-Cost Change
  • Status: Pending → Approved / Rejected
  • AI Feature: Click "🤖 AI Impact Analysis" for cumulative impact assessment

11. RFIs

Track Requests for Information from submission to response.

  • Pre-populated RFIs from SmartPlans import
  • Track status: Open → Answered → Closed
  • AI Feature: "🤖 AI Smart RFIs" drafts professional questions automatically

12. Daily Log

Record daily field reports including:

  • Weather conditions and temperature
  • Crew size and hours worked
  • Work performed (with specific detail)
  • Delays, safety incidents, visitor log
  • Materials received and installed
💡 Pro Tip: Be specific in daily logs — mention room numbers, equipment models, and cable counts. The AI uses your logs for productivity analysis and progress reports.

13. Daily Status Report

The 📧 Daily Report module generates formatted status reports for emailing to stakeholders.

How to Generate a Report

  1. Open a project and click "📧 Daily Report" in the sidebar
  2. Select the report date and enter the recipient email address
  3. Click "📊 Generate Report"
  4. The report appears below with full project status
  5. Click "📋 Copy to Clipboard" to paste into any email, or
  6. Click "📧 Email Report" to open your email client with the report pre-filled

Report Contents

SectionWhat It Shows
📊 Project SummaryContract value, overall progress %, health status, tasks complete
💰 Budget StatusMaterial and labor: budgeted vs. actual with % used and traffic lights
📐 Phase ProgressEach WBS phase with progress %, material spend, and labor hours
🏢 InfrastructureLocation statuses with equipment and cable run counts

14. Punch List

Track deficiency items from walkthrough to closeout.

  • Assign items by location, discipline, and priority
  • Track status: Open → In Progress → Completed → Verified
  • AI Feature: "🤖 AI Priority Plan" ranks items by criticality

16. Troubleshooting

IssueSolution
Can't log inContact your Admin for credentials or password reset
Can't see delete buttonOnly Admin role can delete projects
Budget fields are locked (🔒)Only Admin/Ops Mgr can edit budgets. PMs can edit actuals only.
Import failsEnsure the file is a valid SmartPlans JSON export. Check browser console for errors.
AI analysis failsEnsure you have project data (SOV, logs, etc.) for the AI to analyze. Empty projects return errors.
Daily report won't emailThe email button uses mailto: — you need an email client configured on your device.
Data not savingCheck your internet connection. SmartPM requires connectivity to save to the cloud database.
Project list emptyCreate a project or import from SmartPlans. Ensure you're logged in with the correct account.
